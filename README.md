## 🚀 What is Cali Monitor?

Cali Monitor is a desktop app for Windows that helps you view public information for Santiago de Cali in one place. It brings together news, contract data from SECOP II, and activity by neighborhood in a single panel.

It is built for people who want a clear view of what is happening in the city without opening many sites. You can use it to check media news, sort items by municipal area, and follow local activity from one screen.

## ✨ What you can do

- View local news from sources like Google News, Q'Hubo Cali, Occidente, 90 Minutos, and the Cali city hall
- Sort news by city area, such as Security, Health, Education, Mobility, and Culture
- See SECOP II contract activity in one place
- Track activity by neighborhood
- Review public information in a layout made for quick reading
- Use a single panel instead of switching between many pages

## ❓ Troubleshooting

### The app does not open

Try these steps:

1. Make sure the download finished
2. Extract the ZIP file if you downloaded one
3. Run the `.exe` file, not a shortcut to a missing file
4. Right-click the app and choose **Run as administrator**
5. Restart your PC and try again

### The screen is blank

Try this:

1. Check your internet connection
2. Wait a few seconds for the data to load
3. Refresh the view
4. Close the app and open it again

### News items do not appear

Try this:

1. Confirm that your PC is online
2. Wait for the first load to finish
3. Refresh the section
4. Open another category to check if the issue is only in one view

### The text looks too small

Try changing Windows display settings:

1. Open **Settings**
2. Go to **System**
3. Select **Display**
4. Increase **Scale** or **Text size**
5. Open the app again
